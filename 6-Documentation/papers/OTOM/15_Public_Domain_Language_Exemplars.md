## Executive Summary

This document provides public domain exemplars from archive.org for the 27+ languages documented in the language invariant catalog. These exemplars serve as:
- Training data for invariant detection algorithms
- Reference texts for compression analysis
- Baseline corpora for cognitive load measurement
- Historical language samples for evolutionary analysis

## Usage Guidelines

### For Invariant Detection

1. **Text preprocessing:** Remove formatting, normalize encoding
2. **Language identification:** Use ISO codes from catalog
3. **Invariant extraction:** Apply language-specific detection methods
4. **Compression analysis:** Measure compressibility against baseline

### For Cognitive Load Measurement

1. **Baseline selection:** Use English as reference (CL-I1)
2. **Load calculation:** Apply severity weights from catalog
3. **Cross-linguistic comparison:** Normalize by text length
4. **Evolutionary analysis:** Track changes across time periods

### For Training Data

1. **Corpus construction:** Combine fiction and non-fiction exemplars
2. **Balancing:** Ensure equal representation across genres
3. **Validation:** Use held-out sets for testing
4. **Version control:** Track archive.org identifiers for reproducibility

## Limitations

**Resource constraints:**
- Academic funds at negative balance limited access to paid databases
- Exemplars collected from freely available archive.org resources only
- Some languages rely on grammar references rather than fiction due to availability

**Copyright restrictions:**
- Many languages have limited public domain materials
- Post-1923 works may not be available
- Translation vs. original language distinction

**Quality issues:**
- OCR errors in digitized texts
- Incomplete collections
- Variant editions and translations

**Coverage gaps:**
- Some languages have no public domain exemplars
- Oral traditions not well-represented
- Contemporary works excluded
